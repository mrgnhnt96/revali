# Revali Swagger Annotations

Runtime annotations for the [`revali_swagger`][revali-swagger] construct. Use them to customize generated OpenAPI 3.0 output with API metadata, summaries, tags, responses, and custom schema types.

## Documentation

See the [annotations reference](https://www.revali.dev/constructs/revali_swagger/annotations) for usage examples.
